# AVS Replace

Extension that lets you pick a file, enter find/replace pairs, and writes a new file in the same folder with prefix `AVS-<basename>`.

## Usage

- Press `F5` to launch Extension Development Host.
- Run command `AVS: Open Replace Panel`.
- Click "Choisir le fichier…" to select a file.
- Add pairs of words to find and replacements.
- Click "Générer le fichier AVS" to create the output.

## Build

```powershell
npm install
npm run build
```
